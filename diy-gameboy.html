DIY Gameboy (Bouncing Ball Game)

A compact handheld game console built using Arduino Pro Mini, buttons for input, and a small display. The device runs a simple Bouncing Ball game and demonstrates embedded programming, UI control, and compact hardware integration.

Arduino Pro Mini Display Buttons Portable Game Logic
Code (GitHub)

Key Features

  • Bouncing ball game with simple physics/score logic.
  • Button controls for navigation and gameplay (left/right/select etc.).
  • Portable build on a compact perfboard/proto board.
  • Clean layout with headers/pins for easy maintenance.

How It Works

The Arduino Pro Mini reads button inputs and updates the game state (ball position, lane, score). The display refreshes frame-by-frame to animate the bowling ball movement and results.

Buttons → Input Arduino → Logic Display → Output

Hardware & Electronics

Core Components

  • Arduino Pro Mini (main controller)
  • 8x8 LED display
  • Push buttons (inputs)
  • Battery
  • Perfboard for mounting

Notes

  • Use pull-up resistors (internal or external) for stable button input.
  • Keep wiring short for reliability in handheld builds.
  • Future: add enclosure + speaker/buzzer for sound effects.

Display.ino 

int py = 0;
void display(){
    if (game){
      clear();
      cells[padx][0] = 1;
      cells[padx+1][0] = 1;
      cells[padx+2][0] = 1;

      cells[bx][by] = 1;
    }

  clearRegisters();
  py++;
  if (py == 8){
    py = 0;
  }
  setRegisterPin(15 - py, LOW);

  for (int px = 0; px < 8; px++){
    if (cells[px][py] == 1){
      setRegisterPin(px, HIGH);
    }
  }
  writeRegisters();
}

void clearRegisters() {
  for (int i = (numOfRegisterPins) / 2; i>= 0; i--){
    registers[i] = LOW;
  }
  for (int i = numOfRegisterPins - 1 ; i>= (numOfRegisterPins) /2; i--){
    registers[i] = HIGH;
  }
}

void writeRegisters() {

  digitalWrite(RCLK_Pin, LOW);

  for (int i = numOfRegisterPins - 1; i >= 0; i--){
    digitalWrite(SRCLK_Pin, LOW);

    int val = registers[i];
    
    digitalWrite(SER_Pin, val);
    digitalWrite(SRCLK_Pin, HIGH);

  }
digitalWrite(RCLK_Pin, HIGH);
}

void setRegisterPin(int index, int value){
  registers[index] = value;
}
